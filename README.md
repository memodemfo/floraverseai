# FloraVerse AI

FloraVerse AI is an AI-powered smart gardening app for plant disease detection, plant tracking, smart care recommendations, and predictive gardening insights with an immersive futuristic UI.

## Features

* AI plant disease detection
* Smart plant care recommendations
* Plant lifecycle tracking
* Harvest estimation
* Watering reminders and alerts
* AI-powered gardening assistant
* Predictive plant health analytics
* Cloud database integration with Supabase
* Cinematic nature-inspired UI/UX

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Framer Motion

### Backend

* FastAPI
* TensorFlow
* Supabase

## Backend APIs

* `POST /predict` → AI plant disease prediction
* `POST /add-plant` → Add plant to collection
* `GET /plants` → Fetch all plants
* `GET /plant-alerts` → Smart watering alerts

## Vision

FloraVerse AI combines artificial intelligence, smart agriculture, and immersive design to create a futuristic gardening ecosystem that feels calm, intelligent, and alive.
